01 · Direction & principles

Jade editorial — calm, trustworthy, money-serious

Warm cream surfaces, deep jade rails, gold for statement emphasis, Newsreader serif headlines over a Public Sans UI, and a monospaced column wherever numbers must line up. Note this intentionally overrides the bound "Modernist" design system — the DS-adherence linter will flag every file; that's expected. Four rules to keep it coherent:

Depth is the spine
Every surface is Glance → Working → Deep, shown as underline tabs. Homes are Glance; cores are Working; drill-downs are Deep.
Trust is visible
A freshness badge sits in every top bar; every report carries an as-of date and an Export. Numbers you can't trust are worse than none.
Accent = attention
Jade carries the page; gold/amber flags the ask and drift; red is reserved for aged/error. Green dots mean healthy.
Default-deny
If no role grants a surface, it's blocked — enforced server-side in the DAL, not just hidden in the UI. External partners never touch internal grids.

05 · Roles, i18n, dark & data traps

Roles → access

{{ r.name }} {{ r.sees }}

Internal staff start at full access; the admin can shrink it in the permission grid. External roles are hard-gated to their own rows. Admin (Tim, Van) invites/removes and sets per-person grants; all changes are logged.

i18n · dark · traps

Bilingual (next-intl). All copy from message catalogs; en + 中文. Layouts must flex — Chinese runs shorter; keep tabular numbers LTR. Proven on the explorations toggle.
Dark. Palette pinned in Commission Statement Dark. Brighten jade/gold on dark; drop rules to low-opacity light.
Data traps. Connect revenue_amount = GMV; Boost = ad-spend — never sum across products. Product ids are 'connect' / 'boosted' (never 'boost'). All amounts in dollars. Quote the canonical owed figure live from lens_ps_claim.
